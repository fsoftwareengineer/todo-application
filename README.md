# React.js, 스프링 부트, AWS로 배우는 웹 개발 101

도서 정보: http://www.acornpub.co.kr/book/reactjs-springboot

### 코드 다운로드

```
$ git clone https://github.com/fsoftwareengineer/todo-application.git
```

### 백엔드 코드 실행

```
$ cd <프로젝트 디렉터리>/demo
$ ./gradlew bootRun
```

### 프론트엔드 코드 실행


```
$ cd <프로젝트 디렉터리>/todo-react-app
$ npm install
$ npm start

```
